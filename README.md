# 🏆 Sky-Warp

🏆 **Author:** [@USMCsky](https://github.com/USMCsky)

Sky-Warp is a Java plugin project built with Gradle for Spigot-based Minecraft servers.

## Usage

### Build
```bash
./gradlew build
```

### Build on Windows
```bat
gradlew.bat build
```

### Clean build
```bash
./gradlew clean build
```
